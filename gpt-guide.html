The Library of Congress API

  • By using the LoC API, you can get XML transcripts for 104 out of the 144 oral history interviews.
  • Using the same API, you can also get PDF transcripts for an additional 39 of the interviews.
  • The LoC API can also provide access to 13 articles in the collection.
  • Only one oral history interview completely lacks a transcript.

However, the API doesn't return any plain text. The fulltext
